# SuikiWeb

Labo: Mobilité 
Application web, sous angular 2 (Anglar CLI : https://github.com/angular/angular-cli)
Platforme medecin. 

## Infos Pages : 
 "/" : home et connexion 
 "/inscription' : inscription 
 "/patient" : list des patients 
 "/profile/{id} : profile patient
 "/fiche/{id} : fiche patient 

## Importer node mudules 

Run 'npm install' 

## Run le server

Run 'ng serve' -> localhost:42000

## Run les tests unitaires

Run 'ng test'
